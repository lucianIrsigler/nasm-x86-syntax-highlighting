# NASM x86 syntax highlighting

This extention highlights provides syntax highlighting and snippets for the nasm x86 assembly language

The snippets are the following:

|snippets      |
| ------------ |
|NASMforloop   |
|NASMforloop2  |
|NASMhelloworld|
|NASMifloop    |
|NASMmacro     |
|NASMuserinput |
